== Description ==

[WP JavaScript Typograph Button](http://iskariot.ru/wordpress/typo/#typo-js) предназначен для типографирования текстов в блогах на Wordpress перед записью в базу данных, прямо в визаульном редакторе. _На данный момент проходит бета-тестирование плагина_.
